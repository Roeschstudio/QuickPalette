# QuickPalette

![QuickPalette Logo](https://via.placeholder.com/150x150/000000/FFFFFF?text=QP)

A simple and efficient web application that extracts color palettes from images. Upload any image and instantly get a beautiful color palette that you can use for your design projects.

## üé® Features

- **Drag & Drop Support**: Simply drag an image onto the drop zone or click to browse
- **Color Extraction**: Automatically extracts the 5 dominant colors from any image
- **Multiple Formats**: View colors in both HEX and RGB formats
- **Copy to Clipboard**: One-click copying of color codes
- **Export Palette**: Download your color palette as a JSON file
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **No Server Required**: Everything runs in your browser - your images never leave your device

## üì¶ Installation

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn

### Clone the repository

```bash
git clone https://github.com/yourusername/quickpalette.git
cd quickpalette
```

### Install dependencies

```bash
npm install
```

### Run the development server

```bash
npm run dev
```

Open your browser and navigate to `http://localhost:5173` to see the app in action.

### Build for production

```bash
npm run build
```

The built files will be in the `dist` directory.

## üéØ How to Use

1. Open the application in your browser
2. Drag and drop an image onto the designated area, or click to browse and select an image
3. Wait for the app to extract the dominant colors
4. View the extracted color palette
5. Click "Copy HEX" to copy any color code to your clipboard
6. Click "Download JSON" to export the entire palette
