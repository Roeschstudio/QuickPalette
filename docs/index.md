QuickPalette is a web application that extracts color palettes from images. It's built with modern web technologies and runs entirely in the browser, ensuring your images never leave your device.

- **Privacy-focused**: All processing happens in your browser
- **Easy to use**: Simply drag and drop an image
- **Multiple formats**: View colors in HEX and RGB
- **Export options**: Download palettes as JSON
- **Responsive design**: Works on all devices
